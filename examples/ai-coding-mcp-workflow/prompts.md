# Prompt Examples

These prompts are for developers using AI coding tools with Svg/icons API or MCP workflows. They do not require or imply a fake package or fake local MCP server.

## Live REST API Helper Prompts

- "Find a simple search icon and insert it into this React button."
- "Replace this emoji with an accessible SVG icon."
- "Create matching sidebar icons for users, billing, analytics, and settings."
- "Generate a themeable icon component using currentColor."
- "Find icons for a dashboard navigation and explain why each one fits."

## Review Prompts

- "Check whether these selected icons have consistent visual weight and license metadata."
- "Create a short table of icon ids, names, source sets, and license values from this manifest."
- "Suggest better names for these design-system icon exports without changing their source ids."
